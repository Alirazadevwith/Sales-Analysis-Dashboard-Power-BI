Project Overview

This project is a Sales Analysis Dashboard created entirely in Power BI. It focuses on analyzing beverage sales data across different states and cities, highlighting key performance metrics and insights. The dashboard allows interactive exploration of sales trends, profitability, and delivery performance.

Dataset

The dataset includes the following columns:

Retailer – Name of the retailer

Retailer ID – Unique identifier for the retailer

Invoice Date – Date of the invoice

Region / State / City – Location details

Beverage Brand – Brand of the beverage sold

Days to Deliver – Delivery time in days

Delivery Company – Name of the delivery service

Price per Unit – Unit price of the beverage

Units Sold – Number of units sold

Total Sales – Total sales revenue

Operating Profit – Profit generated from sales

Operating Margin – Profit margin percentage


Dashboard Features

Sales by State – Visual representation of total sales across states.

Key Influencers – Identify factors affecting sales and profit margins.

Matrix Visualization – Detailed tabular view for brand-wise sales performance.

AI Q&A – Ask questions about your sales data using Power BI’s AI insights.
